# 🚀 Gianni Toscano | Full-Stack Web Developer Portfolio

Benvenuto nel repository del mio portfolio personale! Questo sito è stato progettato e sviluppato per presentare il mio percorso professionale, il mio tech stack e i progetti realizzati a seguito del completamento del Master Full-Stack Web Developer presso Epicode.

## 🎨 Caratteristiche del Portfolio

- **Design Premium & Asimmetrico:** Layout moderno ottimizzato con un finto terminale IDE nella Hero Section per un forte impatto visivo.
- **Vetrina Progetti Dinamica:** Layout a righe alterne per una lettura fluida ed efficace delle applicazioni di punta.
- **Form di Contatto Avanzato:** Gestione dello stato in tempo reale tramite React `useState` e predisposizione per integrazione API.
- **Mobile First & Responsive:** Completamente ottimizzato per qualsiasi dispositivo grazie a Bootstrap 5.
- **Smooth Scrolling:** Navigazione fluida tra le sezioni con correzione dell'altezza della navbar.

## 🛠️ Tech Stack

### Frontend
- **React** (Funzioni component, Hooks come `useState`)
- **JavaScript (ES6+)**
- **Bootstrap 5** (Layout Grid, Utility di spaziatura e componenti nativi)
- **CSS3** (Animazioni custom, gradienti dinamici e backdrop-blur)

### Backend & Tooling *(Utilizzati nei progetti in vetrina)*
- **Node.js** & **Express**
- **MongoDB** / **Mongoose**
- **Cloudinary** (Gestione e ottimizzazione dei media cloud)
- **Vite** (Build tool ultra-veloce)
- **Git & GitHub**
